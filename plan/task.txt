## BUGS

BUG [open] - salary_distribution query uses column name "salary_is_predicted" directly in SQL
  but the model column is "salary_predicted". the raw SQL in stats.py references the old name.
  need to check migration or just fix the column reference. throws 500 on /stats/salary-distribution
  if there's no salary data — actually might just be silent with empty results, need to verify.

BUG [open] - useJobs hook dep array uses JSON.stringify(filters) which works but causes a
  re-fetch even when filter values are referentially equal. fine for now, revisit if perf
  becomes an issue with large datasets.

BUG [open] - Jobs page: if user navigates to /jobs?q=something and then clears the search
  in the header, the q param stays in the URL but the filter updates. minor UX issue.

BUG [fixed Mar 8] - spacy model path was failing silently on some systems where the venv
  path isn't in sys.path. added try/except OSError in get_nlp() with None fallback.

BUG [fixed Mar 7] - recharts YAxis width was clipping long category names on the horizontal
  bar chart. fixed by setting width={130} on YAxis for JobsByCategory.
